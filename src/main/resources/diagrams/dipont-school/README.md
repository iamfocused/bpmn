### 这套请假流程需要在EIM中维护的数据有
- 员工类型    
- 汇报给（直属领导）
- 直属SLT

### drools    
- input: 天数、人员类型
- output: 相关人员（人事，行政等） 是否需要SLT审批

### 数据库    
- 假期类型`DP_LEAVE_TYPE_`，注意`Name_`字段，确定好就不能修改，不然会影响流程
- 请假记录`DP_LEAVE_RECORDS_`
- 应享有假期`DP_HOLIDAY_BALANCE_`，如调休这样的享有假期目前是加班转换的（加班流程），因为年假目前所有人都一样，所以现在写死的，可不维护。以后如果每个人年休假不一样，需要维护在这张表中
